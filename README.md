
# My React App with Bun and Vite

This is a React application initialized using Bun and Vite. It serves as a template for quickly setting up a modern front-end project with fast build times and optimized performance.

## Features
- React for building the UI
- Bun as the fast JavaScript runtime
- Vite for fast bundling and hot reloading
- ESLint for code linting
- Pre-configured with basic styles and assets
